## 模块一：圆盘结构渲染

### 1.1 内圈（Inner Ring）

- [x] 以鼠标触发点为圆心，渲染内圈圆盘
- [x] 内圈划分为 **8 个等分扇区**（方向：上/右上/右/右下/下/左下/左/左上）
- [x] 每个扇区显示绑定动作的**图标**
- [x] 每个扇区显示绑定动作的**文字标签**（默认显示；有图标时可配置隐藏）
- [x] 空扇区显示为空白/灰色占位状态

### 1.2 外圈（Outer Ring）

- [x] 内圈外侧渲染第二圈
- [x] 默认模式：**8 个扇区**（与内圈方向对齐）
- [x] 可选模式：**16 个扇区**（每个方向再细分为两个子扇区）
- [x] 外圈**默认不显示文字标签**（盲操作设计）
- [x] 鼠标悬停外圈某扇区时，在**圆盘正中心**显示该动作的名称 + 图标

### 1.3 扩展圈（Extended Ring）

- [x] 扩展圈**不在圆盘上渲染**（视觉上不可见）
- [x] 扩展圈触发区域 = 圆盘圆形边界以外的整个屏幕
- [x] 按 8 个方向划分屏幕区域（以圆心为基准的角度判断）
- [x] 松开鼠标时，若鼠标在圆盘外，根据方向判断触发对应扩展圈动作

## 模块二：触发与交互逻辑

### 2.1 标准触发流程

- [x] 监听指定触发鼠标键（中键 / X1 / X2 / 侧滚轮）的**按下事件**
- [x] 判断"按下后是否立即移动鼠标"：
  - [x] **有移动** → 触发轮盘弹出
  - [x] **无移动** → 视为普通按键，透传给系统
- [x] 轮盘弹出位置 = 鼠标按下时的坐标（圆心）
- [x] 按住状态下，高亮鼠标当前指向的扇区
- [x] **松开触发键** → 执行高亮扇区绑定的动作 → 关闭轮盘

### 2.2 超时自动取消

- [x] 支持配置"超时时间"（0 = 不超时）
- [x] 超时后自动关闭轮盘，不执行任何动作

### 2.3 重复触发（F1 键）

- [x] 轮盘展开状态下，按 **F1（可自定义键）** 立即触发当前高亮扇区动作
- [x] 触发后轮盘**不关闭**，可继续多次按 F1 重复触发
- [x] 按 F1 触发时轮盘保持高亮状态不变
- [x] 支持在设置中将重复触发键改为其他键
- [ ] 支持配置"按特定键时执行不同动作"（v1.28.16+，可选实现）

### 2.4 另类触发（虚拟键 V1，可选）

- [ ] 支持"先按触发键，再按住右键移动"的二段触发方式
- [ ] 触发键按下后 **1 秒内**唤出有效，超时状态失效
- [ ] （此功能为 2022 年 6 月后版本支持，可标注为可选实现）

## 模块三：屏幕边缘适配

- [x] 检测轮盘圆心是否过于靠近屏幕边缘
- [x] **模式 A（不限定）**：圆心就在鼠标位置，部分扇区可能超出屏幕
- [x] **模式 B（限定屏幕）**：自动将圆心向屏幕内偏移，确保整个轮盘都在屏幕内
- [x] **模式 C（限定+移指针）**：在模式 B 基础上，同时把鼠标指针移动到新圆心位置

## 模块四：动作绑定与场景管理

### 4.1 全局场景

- [x] 支持为每个扇区位置（内圈×8 + 外圈×8/16 + 扩展圈×8）绑定动作
- [x] 全局场景为**默认兜底**，任何软件下均生效

### 4.2 软件特定场景（上下文感知）

- [x] 支持创建**基于进程名的软件场景**
- [x] 软件场景中设置的扇区 → **覆盖**全局场景同位置设置
- [x] 软件场景中未设置的扇区 → **回退**到全局场景设置
- [x] 运行时自动检测当前活动窗口进程名，选择对应场景

### 4.3 动作类型支持

- [x] 发送快捷键（模拟键盘组合键）
- [x] 模拟输入文本/按键序列（受输入法影响）
- [x] 粘贴内容（不受输入法影响）
- [x] 打开文件 / 文件夹 / 网址 / 系统命令（支持环境变量展开）
- [x] 运行已有 Quicker 动作（引用方式，非复制）
- [x] 发送文本内容（支持从本地文件读取，`FILE:路径` 格式）
- [x] 插入日期时间（支持 `{0:格式字符串}` 替换符）
- [x] 动作参数传递（同一动作通过不同扇区传入不同参数）

### 4.4 动作引用机制

- [x] 扇区存储的是动作的**引用链接**，不是动作副本
- [ ] 原始动作被删除时，引用失效，需有错误提示
- [ ] 管理界面支持拖放绑定动作

- [x] 轮盘尺寸调整
- [x] 超时取消时间配置（0 = 不超时）
- [x] 开关：允许外圈 16 格模式
