## 角度计算公式

### 扇区索引到角度范围

对于N个扇区（8格模式N=8，16格模式N=16）：
- 每个扇区角度跨度 = 360° / N
- 分割线起始角度 = 292.5°
- 扇区i的起始角度 = 292.5° + (i × 360° / N) （模360°）
- 扇区i的结束角度 = 起始角度 + 360° / N

### 角度到扇区索引

给定角度angle（0~360°）：
```
relAngle = angle - 292.5°
if relAngle < 0: relAngle += 360
sectorIndex = floor(relAngle / (360° / N))
```

## 代码实现要点

1. **点击检测**：在 OnMouseLeftButtonDown 中根据鼠标位置计算角度和距离，判断所在环和扇区
2. **高亮绘制**：在 OnRender 中根据 _highlightRing 和 _highlightSector 判断是否用 C_Sel 颜色绘制
3. **扇区映射**：使用扇区数组和起始角度数组来绘制和检测
