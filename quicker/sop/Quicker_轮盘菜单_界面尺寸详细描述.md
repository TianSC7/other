# Quicker 轮盘菜单 — 界面详细描述（含尺寸）

> 本文档专注于"轮盘弹出后在屏幕上的视觉界面"，包含每一层的几何尺寸、元素布局、颜色状态、字体大小等所有细节，供 AI 实现时精确还原。

---

## 一、整体形态

轮盘是一个**以鼠标触发点为圆心**的圆形叠层菜单，整体由以下区域构成：

```
屏幕
└── 轮盘窗口（无边框透明窗口，覆盖整个屏幕或局部区域）
    ├── 半透明圆形背景层（遮罩）
    ├── 内圈（第一圈，紧靠圆心）
    │     └── 8 个扇形格子
    ├── 外圈（第二圈）
    │     └── 8 或 16 个扇形格子
    ├── 圆心区域（中心信息显示区）
    │     └── 当前高亮动作的图标 + 名称（悬停外圈时显示）
    └── 扩展圈（不渲染，逻辑区域）
```

---

## 二、尺寸体系说明

Quicker 的轮盘大小使用**逻辑单位**（与系统 DPI 缩放联动），设置项名称为"轮盘尺寸"。

### 默认轮盘尺寸参考值（DPI 100% 环境，默认设置）

| 区域 | 逻辑单位 | 实际像素（100% DPI） |
|------|---------|-------------------|
| 圆心死区半径（中心不响应区） | ~20 逻辑单位 | ~20 px |
| 内圈内半径（从圆心到内圈内边） | ~20 逻辑单位 | ~20 px |
| 内圈外半径（内圈外边） | ~80 逻辑单位 | ~80 px |
| 外圈外半径（整个可见圆盘半径） | ~160 逻辑单位 | ~160 px |
| 整个轮盘可见直径 | ~320 逻辑单位 | ~320 px |
| 扩展圈触发范围 | 超出 160 px 的整个屏幕范围 | — |

> ⚠️ **注意**：以上数值为社区实测估算值（DPI 100%，默认尺寸设置），官方文档未公布具体像素值。轮盘尺寸设置项为相对值，实际渲染像素 = 设置值 × DPI缩放系数。
>
> 高 DPI 屏幕（如 125%、150%、200%）下所有数值等比放大。

### 尺寸比例关系（相对比例，与设置值无关）

```
圆心区域 : 内圈宽度 : 外圈宽度 ≈ 1 : 3 : 4
（即内圈宽度约为外圈宽度的 75%）
```

---

## 三、圆心区域

### 形态

- 位于轮盘最中心
- 形状：**圆形**，半径约 20 px（默认尺寸）
- 背景：半透明深色圆形（通常为半透明黑色或跟随主题）
- 平时（鼠标在内圈）：**显示空白** 或 显示轮盘 Logo 图案（主题相关）
- 鼠标悬停外圈扇区时：**显示该扇区动作的图标（居中）+ 动作名称文字（图标下方）**

### 圆心信息显示区尺寸

| 元素 | 尺寸 |
|------|------|
| 圆心区域直径 | ~40 px（与内圈内半径一致） |
| 图标大小（悬停显示时） | ~20×20 px |
| 动作名称字体大小 | ~11–12 px，居中对齐 |
| 图标与文字间距 | ~2–4 px |

---

## 四、内圈（第一圈）

### 几何形状

- 形状：**环形**，被 8 条放射线等分为 8 个扇形
- 内半径：~20 px（从圆心到内圈内边）
- 外半径：~80 px（内圈外边）
- 宽度（环宽）：~60 px
- 每个扇形圆心角：**45°**（360° ÷ 8）

### 8 个方向对应角度（以正上方为 0°，顺时针）

| 方向 | 中心角 | 范围 |
|------|--------|------|
| 上（N） | 0° | -22.5° ~ 22.5° |
| 右上（NE） | 45° | 22.5° ~ 67.5° |
| 右（E） | 90° | 67.5° ~ 112.5° |
| 右下（SE） | 135° | 112.5° ~ 157.5° |
| 下（S） | 180° | 157.5° ~ 202.5° |
| 左下（SW） | 225° | 202.5° ~ 247.5° |
| 左（W） | 270° | 247.5° ~ 292.5° |
| 左上（NW） | 315° | 292.5° ~ 337.5° |

### 内圈格子内的元素布局

```
┌──────────────────────────┐
│        [图标 24×24 px]    │  ← 居中偏上
│      [动作名称文字]        │  ← 图标下方，字体约 11 px
└──────────────────────────┘
```

- 图标尺寸：**24×24 px**（或跟随主题/DPI 缩放）
- 文字：动作名称，字体大小约 **11 px**，单行，超长截断显示省略号
- 图标与文字的组合整体在扇形中**居中对齐**（沿扇形中心线方向）
- 无图标时：只显示文字，字体略大（约 12 px）
- 有图标且开启"有图标时隐藏文字"：只显示图标，图标略大（约 28×28 px）

### 内圈颜色状态

| 状态 | 背景色 |
|------|--------|
| 空扇区（未设置动作） | 透明 / 极浅灰（几乎不可见）|
| 有动作（未高亮） | 半透明白色或浅灰（跟随主题）|
| 鼠标悬停高亮 | 高亮填充色（默认：浅蓝或白色加深，跟随主题）|
| 分隔线 | 细线，颜色比背景略深，宽度约 1 px |

---

## 五、外圈（第二圈）

### 默认模式（8 格）

- 内半径：~80 px（紧接内圈外边）
- 外半径：~160 px（轮盘最外边）
- 宽度（环宽）：~80 px
- 每个扇形圆心角：**45°**，方向与内圈完全对齐

### 16 格模式（开启设置后）

- 内外半径不变
- 每个扇形圆心角变为 **22.5°**（360° ÷ 16）
- 相邻两格之间新增分隔线
- 每个大方向（如"右"方向）被细分为"右偏上"和"右偏下"两个子格

### 外圈格子内的元素布局

- **默认不显示文字和图标**（盲操作设计）
- 格子内**为空**，仅有淡色背景填充和分隔线
- 鼠标悬停时：格子高亮，同时在**圆心区域**（而不是格子上）显示动作名称 + 图标
- 可选：有动作的格子显示一个小圆点或极小图标作为"有内容"的提示（主题相关，非所有主题均有）

### 外圈颜色状态

| 状态 | 背景色 |
|------|--------|
| 空扇区 | 完全透明 |
| 有动作（未高亮） | 极淡填充（半透明，仅作区域感知用）|
| 鼠标悬停高亮 | 高亮填充色（与内圈高亮色一致或略淡）|

---

## 六、整体轮盘背景层

- 形状：圆形（与外圈外半径一致，直径约 320 px）
- 背景颜色：**半透明**，不能为全透明（全透明会导致鼠标事件穿透无法检测）
- 官方提示：最接近"全透明"的可用值为 `#01FFFFFF`（Alpha=1/255，视觉上近似透明）
- 默认主题：半透明白色 `rgba(255,255,255,0.85)` 或半透明深色（深色主题）
- 边缘：无明显边框，或有极细投影（约 4 px 模糊半径）

---

## 七、扩展圈（第三圈，逻辑区域，通常不可见）

- 触发区域：圆盘边界（160 px）以外 → 整个屏幕的其余区域
- 判断方式：松开鼠标时，计算鼠标与圆心的角度，确定触发哪个方向
- 角度划分：与内外圈完全相同，8 方向各 45°
- 正常情况下**无任何视觉渲染**
- 特殊模式（非正常滑动触发）下：可展示第三圈格子，此时外圈再外层出现第三圈可见环，尺寸继续向外延伸约 80–100 px

|------|--------|--------|------|--------|

## 九、图标与字体规格

| 元素 | 规格 |
|------|------|
| 内圈动作图标 | 24×24 px（标准），有图标隐藏文字时约 28×28 px |
| 外圈圆心显示图标 | 20×20 px |
| 内圈文字大小 | 约 11–12 px |
| 圆心文字大小 | 约 11 px |
| 文字字体 | 跟随系统 UI 字体（中文通常为微软雅黑 / 系统默认） |
| 文字颜色 | 深色主题下白色，浅色主题下深灰色 |
| 图标形式 | PNG / SVG / 系统提取图标，支持透明背景 |

## 十、坐标系与方向判断算法

```
以圆心 (cx, cy) 为原点：
dx = mouse.x - cx
dy = mouse.y - cy  （注意：屏幕 Y 轴向下为正）

angle = atan2(dx, -dy)  // 以正上方为 0°，顺时针为正
// 结果范围：-180° ~ +180°，需转换为 0° ~ 360°
if angle < 0: angle += 360

sector_index = floor((angle + 22.5) / 45) % 8
```

方向索引对应：

| 索引 | 方向 |
|------|------|
| 0 | 上（N） |
| 1 | 右上（NE） |
| 2 | 右（E） |
| 3 | 右下（SE） |
| 4 | 下（S） |
| 5 | 左下（SW） |
| 6 | 左（W） |
| 7 | 左上（NW） |

圈判断：

```
dist = sqrt(dx² + dy²)

if dist <= 20:       → 圆心死区，不触发任何动作
if dist <= 80:       → 内圈
if dist <= 160:      → 外圈
if dist > 160:       → 扩展圈（按角度判断方向）
```

## 十一、动画规格

| 动画 | 描述 | 时长（参考）|
|------|------|------------|
| 弹出动画 | 从圆心向外缩放展开（scale 0.3→1.0）+ 淡入（opacity 0→1）| ~120 ms |
| 关闭动画 | 缩放收起（scale 1.0→0.5）+ 淡出 | ~80 ms |
| 扇区高亮切换 | 填充色即时切换，无过渡动画（响应速度优先）| 即时 |
| 圆心文字切换 | 淡入显示新动作名称 | ~60 ms |

> 以上动画时长为社区观测估算值，官方未公布精确数值。

## 十二、主题与颜色变量（以默认浅色主题为例）

| 变量 | 默认值（浅色主题）| 深色主题 |
|------|-----------------|---------|
| 轮盘背景色 | `rgba(255,255,255,0.88)` | `rgba(40,40,40,0.90)` |
| 分隔线颜色 | `rgba(0,0,0,0.12)` | `rgba(255,255,255,0.15)` |
| 普通格子背景 | `rgba(255,255,255,0.0)` | `rgba(255,255,255,0.0)` |
| 高亮格子背景 | `rgba(30,120,255,0.20)` | `rgba(80,160,255,0.25)` |
| 文字颜色 | `#333333` | `#eeeeee` |
| 圆心背景 | `rgba(255,255,255,0.70)` | `rgba(50,50,50,0.80)` |

> ⚠️ 以上颜色值为参考估算，实际颜色受用户安装的**外观主题**影响，可通过 Quicker 外观库完全自定义。外观是独立的样式配置，可覆盖所有默认颜色。
