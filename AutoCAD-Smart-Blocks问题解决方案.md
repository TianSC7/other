# AutoCAD 2024+ Smart Blocks 智能块放置问题解决方案

## 🔍 根本原因：Smart Blocks（智能块放置功能）

块在插入/粘贴时自动旋转，相关线条被高亮为**黄色**，这正是 **Smart Blocks 功能开启**的表现。这是 AutoCAD 2024 引入的新功能。

Smart Blocks 的智能放置会根据你**之前在图纸中放置该块的位置**来提供建议。例如你之前把一个块放在墙角，下次再插入同一个块靠近类似的墙角时，AutoCAD 会自动调整它的**位置、旋转角度和比例**来匹配之前的实例——这就是为什么块会"自己旋转"。

---

## 🟡 黄色线条是什么？

当 Smart Blocks 触发建议时，相关的墙线/边线会**高亮显示（即黄色）**，同时块的位置和旋转会自动对齐到建议位置。你可以点击接受建议，按 **Ctrl** 切换其他建议，或者把光标移开来忽略当前建议。

---

## 🎯 为什么捕捉会"漂移"？

Smart Blocks 的自动放置会**覆盖你的正常 OSNAP 捕捉点**，强制把块吸附到它"认为"合适的位置，导致看起来捕捉点在漂移，插入点不受你控制。

---

## ✅ 解决方法

### 方法一：临时关闭建议（插入时按住）

按住 **Shift+W** 或 **Shift+[** 插入块，可以临时屏蔽智能放置建议。

### 方法二：永久关闭 Smart Blocks 放置建议

在命令行输入以下系统变量：

```
AUTOPLACEMENT
```

将值设为 **0**（关闭）即可彻底禁用自动放置建议。

或者：

```
PLACEMENTSWITCH
```

将 `PLACEMENTSWITCH` 设为 0，可以关闭默认显示的放置建议。

### 方法三：工具选项板块设置旋转提示

如果你想在从工具选项板插入块时**手动控制旋转角度**，可以右键点击工具选项板中的块图标 → 属性 → 将"**旋转提示（Prompt for rotation）**"设置为"是（Yes）"，然后用点击而非拖拽方式插入块。

---

## 📊 现象与原因对照表

| 现象 | 原因 |
|------|------|
| 黄色高亮线条 | Smart Blocks 激活了放置建议，高亮参考对象 |
| 块自动旋转 | Smart Blocks 根据历史放置记录自动匹配角度 |
| 捕捉漂移 | Smart Blocks 劫持了插入点，强制对齐到建议位置 |
| 触发时机 | 从工具选项板用 EXECUTETOOL 插入块时 |

---

## 🚀 快速修复

**最快的修复**：命令行输入 `AUTOPLACEMENT` → 回车 → 输入 `0` → 回车，问题立刻消失。

---

## 💡 额外提示

- 这个功能主要是为了提高重复性绘图工作的效率
- 如果你经常需要插入相同类型的块，Smart Blocks 可以节省大量时间
- 建议在熟悉这个功能后再决定是否完全禁用
- 可以通过设置不同的系统变量来微调 Smart Blocks 的行为
